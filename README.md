# Plataforma Digital de Denuncias (Full Stack)

Plataforma web para que ciudadanos registren denuncias y autoridades gestionen su estado y seguimiento.

## Funcionalidades clave
- Registro e inicio de sesión
- Autenticación con JWT y control de roles (ciudadano / autoridad)
- Crear y listar denuncias
- Cambiar estado y registrar seguimiento
- Subida y visualización de evidencias (imágenes)
- Estadísticas generales para panel de autoridad

## Tecnologías
**Frontend:** React + Bootstrap (+ Chart.js)  
**Backend:** Node.js + Express + JWT + bcrypt  
**Base de datos:** MySQL (Railway)  
**Deploy:** Backend en Render, Frontend en Firebase Hosting  
**Colaboración:** GitHub (trabajo en equipo y control de versiones)

## Trabajo en equipo (2 integrantes)
El proyecto fue desarrollado por **2 personas** y **ambos trabajamos tanto en el frontend como en el backend**, apoyándonos en cada módulo (no se separó el trabajo por completo).  
Usamos GitHub para coordinar avances, integrar cambios y corregir errores en conjunto.
